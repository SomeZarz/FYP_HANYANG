# FYP_HANYANG
Final Year Project for 4th year of Data Science major in Hanyang University. 

## Instructions
First, clone the repo
'''
git clone https://github.com/SomeZarz/FYP_HANYANG
'''

Then, start the install script for environment setup
'''
cd FYP_HANYANG
sh install.sh
'''

Proceed with setting up dir for data
'''
mkdir data
'''
